# labelImg to CSV
LabelImg xml To Google-Cloud data labeling csv file

## Usage
Go to the project folder and...
```bash
$ git clone https://github.com/serhankilicarslan/labelImg-to-csv
$ cd labelImg-to-csv
$ npm install
$ node index.js
```
